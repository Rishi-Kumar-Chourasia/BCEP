Behind BMSCE Events

Exploring the full-stack architecture, security protocols, and creative technologies powering our campus event portal.

HTML5 Structure

The semantic backbone of our portal, providing the accessible skeleton for both Student and Moderator dashboards. It ensures a clean hierarchy for our forms, tables, and event modal displays.

Advanced CSS3

Powers our visual identity using Glassmorphism effects, complex gradients, and flexbox/grid layouts. It handles the responsive design that adapts our "infinite slider" and cards to mobile screens.

Vanilla JavaScript

Drives all client-side interactivity without heavy framework overhead. It manages asynchronous API fetch calls, dynamic DOM updates for filters, and the real-time typewriter effects.

Node.js Runtime

The high-performance, event-driven environment executing our backend code. It efficiently handles concurrent connections during peak registration times, ensuring the server never hangs.

Express Framework

Streamlines our backend routing and middleware integration. It securely manages API endpoints for authentication, event CRUD operations, and student profile retrieval.

MongoDB Atlas

Our scalable NoSQL database storing complex data structures. It maintains collections for Users, Events, and Clubs, enabling fast queries for filtering events by date or category.

SMTP / Nodemailer

The engine behind our communication system. It handles the secure delivery of OTPs for account verification and sends automated confirmation emails upon successful event registration.

JWT Auth

JSON Web Tokens provide stateless, secure authentication. They maintain user sessions across page reloads and protect sensitive Moderator routes from unauthorized student access.

Vanta.js / Three.js

Leverages WebGL to render the immersive, interactive backgrounds (like the Fog and Net effects) you see across the site, adding a modern aesthetic layer to the user experience.

Ngrok Tunneling

Used in our development pipeline to securely expose our local backend to the internet, facilitating the testing of webhooks and external API integrations in real-time.

Git Version Control

Tracks every line of code, enabling modular development and safe rollbacks. It allows our team to collaborate seamlessly on different features like the dashboard and login systems.

01

System Architecture

Designed as a centralized hub bridging Students and Club Moderators. The architecture separates concerns, ensuring that admin capabilities are strictly isolated from the general student interface.

Security First

Utilizing Bcrypt for password hashing and sanitized inputs to prevent injection attacks.

Optimized Data

Efficient MongoDB aggregations for fast loading even with large event lists.

CI/CD Pipeline

Automated workflows ensure code quality before deployment.

Robust API

Custom REST endpoints handling everything from Login to Profile updates.
